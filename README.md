# Тестовое задание аналитика (БА/СА)

## Описание
Решение тестового задания, включающего:
- BPMN модель процесса выдачи оборудования
- Требования к функциональности маркетплейса
- REST API и алгоритм регистрации пользователя

## Структура

### 📌 Задача 1
Папка: task-1-bpmn  
Содержит BPMN диаграмму процесса

### 📌 Задача 2
Папка: task-2-requirements  
Содержит User Story, Use Case и требования

### 📌 Задача 3
Папка: task-3-api  
Содержит OpenAPI спецификацию и алгоритм backend
